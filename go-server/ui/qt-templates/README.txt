Script Manager:

- How best to present information?

- Chat window!

- Table with one row for script, Open UI screen for script through button:
    - Pros: Clean design for quick running of script
    - Cons: Can't switch / keep track of multiple scripts

- Show only running scripts. Single UI, but slider menu. 
  To start a script, choose from menu, or seach and start above.
  Stack all script screens one below the other. 
   - Pros: Quickness of access
   - Cons: Long screen of screens is not so good..


- View as tile blocks of running script screens. Show live status.
